#### "오늘의 복지" 
: AI 챗봇 애플리케이션을 활용한 정부 및 지자체의 혜택 · 복지 · 소식 정보를 제공 및 추천 해주는 서비스 
- 사용자의 자연어 입력을 NLP 처리하여 의도를 파악
- 해당 의도를 크롤링 서버로 보내서 지정된 URL에서 크롤링
- 크롤링 정보를 가공하여 사용자에게 제공 
- 기존의 입력된 개인 정보와 검색 기록을 토대로 비슷한 유형의 혜택 · 복지 · 소식 정보 추천
